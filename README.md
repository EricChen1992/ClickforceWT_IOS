# iOS ClickforceWT
###### Clickforce 客製化打卡App 

* 功能
  * 登入驗證
  * 打卡
  * 歷史打卡記錄
  * 寫入日曆
  * Notification(地區判斷打卡通知/時間判斷上下班通知)
